### ⛅️Project Description <hr/>
> 오늘의 날씨와 기분을 반영해 사용자 맞춤형 콘텐츠를 추천해주는 웹 서비스입니다.<br>
> 지도 및 날씨 API를 통해 오늘의 날씨를 가져오거나, <br>
> 서베이를 통해 사용자의 마음 날씨를 파악하고 이에 맞는 콘텐츠를 추천합니다.
